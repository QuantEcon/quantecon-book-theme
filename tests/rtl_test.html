عبرית

זהו טקסט לדוגמה בעברית לבדיקת תמיכה בכתיבה מימין לשמאל בנושא זה. הטקסט אמור להיות מוצג כראוי עם הכיוון המתאים.

Code Block Test

Code blocks should remain left-to-right:

def hello_world():
    return "Hello, World!"

Math Test

Mathematical expressions should also remain LTR:

E = mc²
